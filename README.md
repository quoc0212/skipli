# Skipli 

## Project structures
```bash
├── server                          # node app
    ├── js                          # controllers
    ├── constants                   # constant files
    └── index.js                    # main file
├── src                             # react app
    ├── components                  # re-use components
    ├── api                         # axios config
    ├── features                    # pages
    ├── constants                   # constant files
    └── app.js
```


## Installation

Install all dependencies with:

```bash
npm install
```

## Run app

Run app with:

```bash
npm start
```
